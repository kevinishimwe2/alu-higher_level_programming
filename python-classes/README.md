# Python - Classes and Objects

This project explores Object-Oriented Programming (OOP) concepts in Python, focusing on classes, objects, attributes, and methods.

## Learning Objectives

- Understanding OOP principles
- Working with classes and objects in Python
- Implementing private, protected, and public attributes
- Using properties with getters and setters
- Data abstraction, encapsulation, and information hiding
- The `__init__` method and instance initialization
- Dynamic attribute creation and the `__dict__` attribute

## Project Tasks

### Task 0: My first square
- Create an empty `Square` class

### Task 1: Square with size
- Add a private instance attribute `size`
- Initialize with a size parameter

### Task 2: Size validation
- Add type and value validation for size
- Size must be an integer >= 0

### Task 3: Area of a square
- Add `area()` method to calculate square area

### Task 4: Access and update private attribute
- Implement property getter and setter for size
- Pythonic way to access private attributes

### Task 5: Printing a square
- Add `my_print()` method to print the square with `#` characters

### Task 6: Coordinates of a square
- Add `position` attribute (tuple of 2 positive integers)
- Modify `my_print()` to respect position offset

## Requirements

- Python 3.8.5
