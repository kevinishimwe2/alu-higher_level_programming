# Python - if/else, loops, functions

This project covers fundamental Python programming concepts including conditional statements, loops, and functions.

## Learning Objectives

- Understanding Python programming fundamentals
- Working with if/else statements
- Using while and for loops
- Creating and using functions
- Understanding variable scope
- Working with the range function
- Using break and continue statements
- Understanding Python code style (pycodestyle)

## Project Files

- `0-positive_or_negative.py` - Checks if a random number is positive, negative, or zero
- `1-last_digit.py` - Prints the last digit of a random number with classification
- `2-print_alphabet.py` - Prints the ASCII alphabet in lowercase
- `3-print_alphabt.py` - Prints the alphabet except 'q' and 'e'
- `4-print_hexa.py` - Prints numbers 0-98 in decimal and hexadecimal
- `5-print_comb2.py` - Prints numbers 00-99 with formatting
- `6-print_comb3.py` - Prints all unique combinations of two different digits
- `7-islower.py` - Function to check if a character is lowercase
- `8-uppercase.py` - Function to convert a string to uppercase
- `9-print_last_digit.py` - Function to print and return the last digit of a number
- `10-add.py` - Function to add two integers
- `11-pow.py` - Function to compute power of a number
- `12-fizzbuzz.py` - Classic FizzBuzz implementation

## Requirements

- Python 3.8.5
- Ubuntu 20.04 LTS
- pycodestyle version 2.7.*
- All files must be executable
- All files must start with `#!/usr/bin/python3`
