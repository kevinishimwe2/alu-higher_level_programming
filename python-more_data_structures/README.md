# Python - More Data Structures: Set, Dictionary

This project contains Python programs demonstrating the use of sets, dictionaries, lambda functions, and map/filter/reduce.

## Tasks

0. **Squared simple** - Compute the square value of all integers of a matrix
1. **Search and replace** - Replace all occurrences of an element in a new list
2. **Unique addition** - Add all unique integers in a list
3. **Present in both** - Return common elements in two sets
4. **Only differents** - Return elements present in only one set
5. **Number of keys** - Return the number of keys in a dictionary
6. **Print sorted dictionary** - Print a dictionary by ordered keys
7. **Update dictionary** - Replace or add key/value in a dictionary
8. **Simple delete by key** - Delete a key in a dictionary
9. **Multiply by 2** - Return new dictionary with all values multiplied by 2
10. **Best score** - Return key with the biggest integer value
11. **Multiply by using map** - Multiply list values using map and lambda
12. **Roman to Integer** - Convert a Roman numeral to an integer

## Key Concepts

- **Sets**: Unordered collections of unique elements
- **Dictionaries**: Key-value pairs for efficient lookup
- **Lambda functions**: Anonymous functions for simple operations
- **Map**: Apply a function to all items in a list
- **List/Dictionary comprehensions**: Concise way to create lists and dictionaries

## Requirements

- All files are interpreted/compiled on Ubuntu 20.04 LTS using python3 (version 3.8.5)
- All files end with a new line
- The first line of all files is exactly `#!/usr/bin/python3`
- Code uses pycodestyle (version 2.7.*)
