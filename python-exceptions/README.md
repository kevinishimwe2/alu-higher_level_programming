# Python - Exceptions

This project contains Python programs demonstrating exception handling.

## Tasks

0. **Safe list printing** - Print x elements of a list using try/except
1. **Safe printing of an integers list** - Print an integer with proper format validation
2. **Print and count integers** - Print only integers from a list, skip other types
3. **Integers division with debug** - Divide two integers with try/except/finally
4. **Divide a list** - Divide two lists element by element with error handling
5. **Raise exception** - Raise a TypeError exception
6. **Raise a message** - Raise a NameError with a custom message

## Key Concepts

- **Exceptions**: Runtime errors that can be caught and handled
- **try/except**: Catch and handle exceptions
- **try/except/finally**: Execute cleanup code regardless of exceptions
- **raise**: Manually raise exceptions
- **Error types**: TypeError, ValueError, IndexError, ZeroDivisionError, NameError

## Requirements

- All files are interpreted/compiled on Ubuntu 20.04 LTS using python3 (version 3.8.5)
- All files end with a new line
- The first line of all files is exactly `#!/usr/bin/python3`
- Code uses pycodestyle (version 2.7.*)
- All files are executable
