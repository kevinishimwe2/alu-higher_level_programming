# Python - More Classes and Objects

This project is part of the ALU Higher Level Programming curriculum, focusing on Object-Oriented Programming (OOP) in Python.

## Project Description

This project implements a `Rectangle` class with various features that demonstrate key OOP concepts in Python, including:

- Private attributes with property getters and setters
- Input validation
- Instance and class methods
- Static methods and class methods
- String representation methods (`__str__` and `__repr__`)
- Destructor method (`__del__`)
- Class attributes
- Object comparison

## Learning Objectives

By completing this project, you will understand:

- What is OOP and "first-class everything"
- The difference between classes, objects, and instances
- Public, protected, and private attributes
- The `self` parameter and instance methods
- Special methods: `__init__`, `__str__`, `__repr__`, `__del__`
- Data abstraction, encapsulation, and information hiding
- Properties vs attributes and the Pythonic way to write getters/setters
- Class attributes vs instance attributes
- Class methods and static methods
- Dynamic attribute creation and the `__dict__` attribute
- How Python finds attributes using the attribute lookup chain
